## CAP Coordinate System

CRF는 내적 의식 상태를 **CAP**이라는 3축으로 표현합니다.

| 축 | 의미 | 예시 질문 |
|---|---|---|
| C — Coherence | 말/의도/감정이 얼마나 일치하는가 | “내가 지금 정말 하고 싶은 말이 맞나?” |
| A — Amplitude | 감정 진폭의 크기 | “지금 감정은 잔잔? 선명? 강렬?” |
| P — Phase | 관점의 방향성 (내향↔중립↔외향) | “나는 지금 안쪽을 보고 있나, 바깥을 보고 있나?” |
