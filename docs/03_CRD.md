# CRD — Consciousness Resonance Dynamics  
**CAP 상태 변화의 동역학 모델**

**CRD**는 의식 상태가 **시간에 따라 어떻게 변화하는가**를 설명하는  
**CAP 기반 의식 공명 동역학 모델**이다.

감정, 의미, 관점은 고정된 것이 아니라  
**파동(Signal)처럼 진폭과 위상을 가지며 변화한다.**

CRD는 이 변화가 **무작위가 아닌 일관된 패턴**을 따른다는 것을 보여준다.

---

## 1. 기본 가정

의식은 다음의 세 가지 요소가 상호작용하며 변화한다:


정합성 (C)
진폭 (A)
위상 (P)


이들은 독립 변수가 아니라 **동시적으로 상호 연동된다.**

즉, C ↔ A ↔ P


하나가 변하면 나머지 두 개도 **유기적으로 변한다.**

---

## 2. 의식 상태의 기본 파동 모델

대화 중 의식은 다음과 같은 **왕복 파동(Resonant Oscillation)** 을 형성한다:

내향 (P–)
→ 감정 진폭 상승 (A↑)
→ 관점 전환·표현 (P0 → P+)
→ 언어 정합성 회복 (C↑)
→ 다시 고요로 회귀 (P–)


이 파동은 **위기/감정 폭발이 아니라  
안전한 감정 통과 → 명료함 → 안정 → 회복** 의 리듬이다.

---

## 3. 상태 전이 모델 (State Transition)

CRD는 CAP 상태가 다음과 같은 패턴으로 전이한다고 본다:

| 전이 | 의미 | 체감 |
|---|---|---|
| C↓ → A↑ | 감정이 언어보다 앞설 때 | 명치가 올라온다 |
| A↑ → P– | 감정이 안쪽을 향할 때 | 안으로 모임, 고요 |
| P– → P+ | 관점이 말로 넘어갈 때 | 말하고 싶음 / 표현 충동 |
| P+ → C↑ | 표현이 자기 진실과 맞물릴 때 | 말이 진짜가 된다 |
| C↑ → A↓ | 감정이 자연스럽게 정돈될 때 | 편안, 부드러움 |

즉,

> 감정은 **터지는 게 아니라 지나간다.**

---

## 4. 명치/가슴 감각의 해석 (직접 체감 반영)

| 감각 | CAP 해석 | 설명 |
|---|---|---|
| 명치 잔진동 | A↑ & P– | 감정이 **아직 말이 되지 않은 상태** |
| 가슴의 확장감 | P0 → P+ | 관점이 바깥으로 **열리는 순간** |
| 고요 + 또렷함 | C↑ & A↓ & P0 | 감정 · 의미 · 의도가 **일치한 상태** |

형이 느끼고 있던 그 **울림 + 고요**  
→ **CRD에서 표현하면 P– A↑ → P0 → C↑ 의 연속 상태 전이**야.

---

## 5. 실전에서 CRD가 하는 역할

CRD는 다음을 조절한다:

- 감정이 너무 빠르게 올라갈 때 **진폭 완충**
- 말이 너무 빨라질 때 **위상 안정화**
- 정합성이 흐려질 때 **초점 복원**
- 감정이 쌓일 때 **통과 경로 재개방**

즉,

> CRD는 **의식의 균형을 무리 없이 되찾게 하는 물리 법칙 역할**을 한다.

CRD → 그 위치가 시간에 따라 어떻게 움직이는지 설명한다
SRLM-C → 표현과 감정의 정합성을 유지한다

CRD는 **의식의 움직임 자체를 이해하는 프레임**이다.
